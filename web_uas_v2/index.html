Lyricaly

Lyricaly is a web app that made for a musician who need an inspiration for creating a lyric based on mood and help them generate a specific lyric with their own keyword. This web app is free to use, and made simple for user friendly experience. We hope that this app can help many people and bring inspiration for every user.
